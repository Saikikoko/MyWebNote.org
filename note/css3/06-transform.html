# transform

transform变换后，仍占据文档变换之前的位置，但是不改变文档的大小

一个很形象的比喻：灵魂出窍

# 2d变换

  1. rotate旋转

    transform:rotate(30deg);/*沿着中心旋转30度*/
transform:rotateX(30deg) rotateY(30deg);/*先沿着x旋转30度，再沿着y轴旋转30度*/
transform:rotate(1,1,90deg);/*同时沿着x、y轴旋转30度*/
transform:rotate3D(1,1,1,30deg);/*同时沿着x、y、z轴旋转30度*/

  2. scale缩放

    默认值是1，表示不缩放

    transform: scale(1);
transform: scale(1.5);
transform: scaleX(2);
transform: scaleY(2);
transform: scale(4,2);
transform: scale(4,4);
transform: scale(4);
transform: scale3D(1,1,1);

  3. translate位移

    使用left，top属性做动画（轮播）会出现卡帧，使用translate则不会

    transform:translate(50px,50px);/*同时向x轴、y轴移动50px*/
transform:translateY(50px);/*向y轴移动50px*/
transform:translateX(50px);/*向x轴移动50px*/
transform:translateY(50px) translateX(50px);/*先向y轴移动50px，再向x轴移动50px*/
/*占文档流位置，不改变文档流大小*/
/*
translateZ(0px) 可以开启硬件加速(多用于移动端)
*/

  4. skew倾斜

    transform:skew(20deg,30deg);/*水平方向倾斜20deg，垂直方向倾斜30deg*/

# 变换基点

  1. transform-origin
    • 默认基点为中心点，可以通过设置坐标值或关键词改变基点
    • 例：transform-origin:left top; 变换基点改为左上角
  2. perspective-origin 景深基点 人的观察点

# 3d变换

  1. 变换风格transform-style 和 景深perspective (近大远小)

    浏览器默认2D变换场景，是无法体现出3D变换样式的，所以在3D变换时要加上这两个属性

    transform-style:preserve-3d

  2. backface-visibility 隐藏背面
