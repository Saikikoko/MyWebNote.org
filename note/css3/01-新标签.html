# 新标签

# 文档声明简化

  1. DOCTYPE

    HTML4需要DTD( document type definition)

    <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.1//EN" "http://www.w3.org/TR/xhtml11/DTD/xhtml11.dtd">

    HTML5不需要DTD，只需要写：

    <!DOCTYPE html>

  2. meta标签规定编码

    html4:

    <meta http-equiv=Content-Type content="text/html;charset=utf-8">

    html5:

    <meta charset="UTF-8">

# 新增标签

  1. 标签结构（重点掌握这几个标签）

    • header 定义文档的头部区域，一般用在顶部

    • article 定义页面独立的文章内容，帖子、博客文章、评论、新闻等可以使用

    • nav 定义导航，替代ul和li定义的导航

    • section 定义一块区域，一般用来写主体内容部分

    • aside 定义侧边栏，侧边信息可以使用

    • figure 定义一块独立的内容，通常用来展示图片及其描述

    • figcaption 定义figure的标题

    • iframe 框架，在页面中创建一个包含网页文档的框架 frameset已经被淘汰

      • <iframe src=''    name=''      frameborder='' ></iframe>

      • src 表示在框架里显示文档的url

      • name 表示框架的名字

      • frameborder 表示框架是否有边框 0表示没有 1表示有边框

    • footer 定义文档的底部，一般以它结尾

    • main定义网页主体部分，一个网页只能有一个main标签

  2. 特殊样式标签

    • mark 突出显示文字

    • meter 刻度百分比

      • <meter value="10" low="30" high="40" min="0" max="70" optimun="37.5"></meter>

      • value:当前刻度值

      • low: 定义度量的值位于哪个点，被界定为低的值。

      • high：定义度量的值位于哪个点，被界定为高的值

      • min：定义最小值。默认值是 0。

      • max：定义最大值。默认值是 1。

      • optimum：定义什么样的度量值是最佳的值

        • 如果该值高于 "high" 属性，则意味着值越高越好。
        • 如果该值低于 "low" 属性的值，则意味着值越低越好

    • progress 进度条

      • <progress value="20" max="100">

    • ruby 内容+注释 rt注释内容 rp不支持时显示

    • wbr 长单词换行的位置

  3. 兼容性不是很好的标签

    • dialog 定义一个对话框
    • details / summary描述某部分的详情，summary为details的标题

  4. 高级标签

    canvas 、svg 、video 、audio等更高级的功能标签

思考：

  1. 为什么要新增这些标签？

    结构标签让语义化的特点更突出，特殊样式标签让内容展示更方便，便于搜索引擎

  2. 低版本不兼容怎么办？

    在低版本浏览器会出现不兼容情况，一般使用JavaScript来兼容

    当浏览器不认识HTML5新标签时，会将其解析为一个自定义标签，这样的标签是没办法加样式的（可以使用IE8进行测试），要解决这些新标签样式问题，也就是要解决自定义标签样式问题，代码如下：

    <!DOCTYPE html>
<html>
    <head>
        <title></title>
        <meta charset="utf-8">
        <!--在样式之前加入这段JS代码-->
        <!--[if lt ie 9]>
            <script>
        		document.createElement("goudan");
        	</script>
        <![endif]-->
        <style type='text/css'>
            .hi{
                color: red;
            }
        </style>
    </head>
    <body>
        <goudan class="hi">hello</goudan>
    </body>
</html>

# HTML5兼容性

HTML5有部分内容兼容到IE9，IE8及以下对H5完全不兼容后面的内容不再考虑此类浏览器

部分内容需要加兼容前缀,这些兼容前缀，都是老外的，国内唯一有内核是TX X5内核:

  • -webkit-常用于兼容chrome浏览器
  • -moz-常用于兼容火狐
  • -o-常用于兼容opera
  • -ms-常用于兼容IE
