# flex弹性盒模型

特点：布局灵活，多变

pc端：传统的布局方式，display+float+position

移动端：常用flex的几个属性进行布局

小程序：基本都是弹性盒模型布局的

# 弹性盒子布局

容器默认存在两根轴：水平的主轴（main axis）（从左向右）和垂直的交叉轴（cross axis）（从上到下）。主轴的开始位置（与边框的交叉点）叫做main start，结束位置叫做main end； 交叉轴的开始位置叫做cross start，结束位置叫做cross end。 项目默认沿主轴排列。单个项目占据的主轴空间叫做main size，占据的交叉轴空间叫做cross size。

display:flex

设置该属性后，子元素默认从左向右排列，且当子元素没有设置宽高时，子元素的宽由内容撑开，高度则继承容器的高度

常用属性：

  • dipaly:flex
  • justify-content:space-between
  • justify-content:space-around
  • flex:1
  • order:number
  • align-items:center

# 容器的属性

# 1.flex-direction 排列盒子方向

row(默认)： 横向从左到右排列（横向 ）。 row-reverse： 对齐方式与row相反。 （倒序排列） column： 纵向从上往下排列（纵向）。 column-reverse：对齐方式与column相反。 （倒序排列）

# 2.flex-wrap 换行方式

控制flex容器是单行或者多行，同时横轴的方向决定了新行堆叠的方向

  • nowrap（默认）： flex容器为单行，不换行，子元素宽度超出容器的宽度，子元素会被等比例压缩
  • wrap： flex容器为多行。超出换行 ，在交叉轴上等比例排列
  • wrap-reverse：反转 wrap 排列。交叉轴方向改为从下到上

# 3.flex-flow:flex-direction flex-wrap

flex-flow：属性是flex-direction属性和flex-wrap属性的简写形式，默认值为（row nowrap）

# 4.justify-content

设置子元素在主轴上的对齐方式

  • flex-start（默认值）： 左对齐
  • flex-end： 右对齐
  • center： 居中
  • space-between： 两端对齐，子元素之间的间隔都相等。(自动设置间距)
  • space-around： 每个子元素两侧的间隔相等。所以，子元素之间的间隔比子元素与边框的间隔大一倍。（自动设置间距）

# 5.align-items

设置子元素在交叉轴上的对齐方式

具体的对齐方式与交叉轴的方向有关，下面假设交叉轴从上到下

  • flex-start： 交叉轴的起点对齐。
  • flex-end： 交叉轴的终点对齐。
  • center： 交叉轴的中点对齐。
  • baseline: 项目的第一行文字的基线对齐。
  • stretch（默认值）：如果项目未设置宽高或设为auto，将沿交叉轴占满整个容器的高度或宽度。

# 6.align-content

属性定义了交叉多根主轴（多行）的对齐方式。如果项目只有一根轴线，该属性不起作用

  • flex-start： 与交叉轴的起点对齐。
  • flex-end： 与交叉轴的终点对齐。
  • center： 与交叉轴的中点对齐。
  • space-between： 与交叉轴两端对齐，轴线之间的间隔平均分布。
  • space-around： 每根轴线两侧的间隔都相等。所以，轴线之间的间隔比轴线与边框的间隔大一倍。
  • stretch（默认值）：等比例占满整个交叉轴。在设置了align-items后无效

多行下的align-content会使align-items失效

# 子元素的属性

以下6个属性设置在子元素上。

  • order 子元素排序
  • flex-grow 子元素拓展比例
  • flex-shrink 子元素缩小比例
  • flex-basis 定义拓展剩余空间前，的交叉轴的宽
  • flex 复合属性
  • align-self 自定义子元素align-items

# 1.order

order属性定义项目的排列顺序。数值越小，排列越靠前，默认为0。可以负值

# 2.flex-grow：<number>

定义项目的放大比例，（默认为0） 即如果存在剩余空间，也不放大

子元素的放大尺寸=父级盒子剩余空间尺寸*子元素的flex-grow属性值 / 所有子元素的flex-grow属性值的和

# 3.flex-shrink

定义了项目的缩小比例，（默认为1），即如果空间不足，该项目将缩小。

如果一个项目的flex-shrink属性为0，其他项目都为1，则空间不足时，前者不缩小。 负值对该属性无效。

# 4.flex-basis

它的默认值为auto。

该属性定义了在分配多余空间之前，项目占据的主轴空间（main size）浏览器根据这个属性，计算主轴是否有多余空间。

说白了就是设置width

# 5.flex

该属性是flex-grow, flex-shrink 和 flex-basis的简写，默认值为(0 1 auto)

该属性有两个快捷值：auto (1 1 auto) 和 none (0 0 auto)。

  • auto(1 1 auto)有剩余空间，则放大，剩余空间不足，则缩小，该属性很常用
  • none(0 0 auto) 失去弹性

# 6.align-self

该属性用于单独设置子元素在交叉轴上的对齐方式

（默认值为auto），表示继承父元素的align-items属性，如果没有父元素，则等同于stretch。
