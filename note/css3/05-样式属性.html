# border-radius

border-radius:50%;
border-top-radius:50%
border-bottom-radius:50%;
border-left-radius:50%;
border-right-radius:50%;
border-radius:50px 10px 30px 40px;

和margin、padding一样的用法

border-radius: x1 x2 x3 x4/y1 y2 y3 y4

(x1,y1)代表左上角的值

(x2,y2)代表右上角的值

# 文本属性

white-space:nowrap

控制文本不换行

/*单行文本超出容器，用···替代文本*/
p{
    overflow：hidden；
    white-space：nowrap；
    text-overflow：ellipsis；
}
/*一般通过后台控制*/

word-wrap:break-word

使用场景：

​ 长单词在第一行显示不下

​ 特点：会到第二行显示，超出的部分，继续到下一行显示

​ 缺点：浪费空间

word-break:break-all

使用场景：

​ 长单词在第一行显示不下

​ 特点：直接在第一行显示，超出的部分，到下一行显示

​ 优点：不会浪费空间
