# 字符串方法和数组

在js中点操作的都是对象，但是有时也会看到.对基础数据类型进行操作如数字和字符串，此时其实.操作的是他们的包装对象。

let a = "阿飞";
a.index = 20;
console.log( a.index );//und
/*包装对象不会保存自定义属性的值，因为每执行完一句语句，包装对象都会重新生成*/
let b = 20;
b.goudan = 5;
console.log(b.goudan);//und

# 字符串方法

String即文本（字符串），字符串方法都不改变原字符串，而是创建并返回一个新的字符串；

创建字符串的三种办法: new String(), String(), 直接量，三种方式可以创建，不同点在于new String()创建的是一个String对象，点操作赋值可以改变元素本身的值

let str = new String("二哈")；
let str1 = String("小柴")；
let str2 = "小强"；
str.length //字符串中字符长度
str[index]//通过下标index获取字符串中的某个字符

  1. # str.charAt(index)

返回指定索引的字符串

let a = "二哈"；
console.log(a.charAt(1));//哈
//a[index] === a.charAt(index)
console.log(a[1]);//哈

//字符串方法不改变原字符串
a[0] = 5;
console.log(a[0]);//二

  1. # str.charCodeAt(index)

返回指定索引的ASCLL编码

let a = "二哈"；
console.log(a.charCodeAt(1));//21704

  1. # String.fromCharCode(unic,unic,unic)

返回指定ASCLL编码的字符

console.log(String.fromCharCode(888,666,777));

  1. # str.substring(start,end )

截取字符串，从start开始，截止到end前，不包含end，如果没有end则从num开始整个查找

如果start比end大，那么该方法在提取子串之前会先交换这两个参数

let str = "看，这个锅又大又圆";
console.log(str.substring(2,4));//这个
console.log(str.substring(4,2));//这个
console.log(str.substring(2));//这个锅又大又圆
console.log(str.substring(-2,2));//看， //start或end为负数则认为是0

  1. # str.slice(start,end )

截取字符串，从start开始，截止到end前，不包含end

和substring用法一样,从左往右截取

start / end可以为负值，负值时，倒着从最后数

start 和 end任何时候都不会交换位置，能截取才有值，截取不到则无值

  1. # str.toLocaleUpperCase()/ str.toLocaleLowerCase()

str.toLocaleUpperCase() 把字符串转换为大写。

str.toLocaleLowerCase() 把字符串转换为小写。

  1. # str.indexOf(value,index )

查找字符串，返回查找字符串首次出现的位置

对大小写敏感！

value 匹配字符

index开始检索的位置, 合法值是 0到 string.length - 1,默认0

匹配失败返回-1

str.lastIndexOf(value,index)从后向前搜索,合法值是0到 string.length - 1,默认string.length - 1匹配失败返回-1

let a = "我觉得阿飞老师是最帅的老师";

//alert(  a.indexOf("老师")  ); //5

//alert(  a.indexOf("老师" , 10)  );//11

alert( a.indexOf("朱雀") ) //-1

以下方法可以用正则（RegExp）代替字符串进行匹配

  1. # str.split( )

str.split(value/RegExp,length-1)方法用于把一个字符串分割成字符串数组, 返回分割后的数组

第二个参数是可选参数，是指定返回数组的长度，最大为str.length-1，不写默认str.length-1

let a = "阿飞，朱雀，心艾，岚岚，丫丫，艾米";
let b = a.split("，");
console.log(b);//[阿飞,朱雀,心艾,岚岚,丫丫,艾米]

  1. # str.match( )

str.match( value/RegExp )查找指定的值，返回匹配的值。未找到返回null.正则可以找一个或多个表达式

  1. # str.search( )

str.search( value/RegExp )返回 检索字符串首次出现的位置;未找到返回-1

# 数组

创建数组的三种办法: new Array(), Array(), [] ，三种方式创建出来都是一样的

let arr = new Array();
let arr1 = Array();
let arr2 = [];

arr.length可以访问数组的长度

创建即指定数组长度Array( length )及 new Array( length ),length是 数字的时候，创建的并不是数组的项，而是数组的长度，项的内容为undefined

let arr = new Array(4);
arr.length //4;

[]通过数组索引，访问值

和字符串不同，数组可以修改指定索引下的值

let arr = [4,5,6,7];
arr[0] = 9;//[9,5,6,7]

arr.indexOf( item )查找项，返回item所在index索引

# 数组的api

和字符串不同，数组的成员函数都是在其原始值上进行操作

  1. # arr.push()

arr.push( item1,item1,….)向数组的尾部添加一个或更多元素，并返回（新长度）

  1. # unshift()

arr.unshift(item1,item1,….)向数组的头部添加一个或更多元素，并返回（新长度）

  1. # arr.pop()

arr.pop() 删除数组的最后一个元素（返回删除对象）

  1. # arr.shift()

arr.shift()删除数组的第一个元素（返回删除对象）

  1. # arr.splice()

arr.splice(index,howmany,item1,…..,itemX)（删除/添加） 元素，然后（只返回删除对象）。

index必需。整数，规定添加/删除项目的索引，可以使用负数,如果是添加，原有元素会往高位移动。

howmany。要删除的项目数量。如果设置为 0，则不会删除项目。不写，默认从index开始删除所有元素

item1, ..., itemX可选。向数组添加的新项目。

let a = ['阿飞',"无虑","小浪"];
a.splice(1);
console.log(a);//["阿飞"]

let b = ['阿飞',"无虑","小浪"];
b.splice(1,1);
console.log(b);//["阿飞","小浪"]

let c = ['阿飞',"无虑","小浪"];
c.splice(1,1,"朱雀");
console.log(c);//["阿飞","朱雀","小浪"]

let d = ['阿飞',"无虑","小浪"];
d.splice(1,0,"朱雀");
console.log(d);//["阿飞","朱雀","无虑","小浪"]

  1. # arr.slice()截取

arr.slice(start,end)方法从已有的数组中返回一个新的数组对象。(不修改原数组)

arr.slice();
// [0, end]

arr.slice(begin);
// [begin, end]

arr.slice(begin, end);
// [begin, end)
  • begin 可选

从该索引处开始提取原数组中的元素（从0开始）。

如果该参数为负数，则表示从原数组中的倒数第几个元素开始提取，``slice(-2)表示提取原数组中的倒数第二个元素到最后一个元素（包含最后一个元素）。

如果省略 begin，则 slice 从索引 0 开始。

  • end可选

在该索引处结束提取原数组元素（从0开始）。slice会提取原数组中索引从 begin 到 end的所有元素（包含begin，但不包含end）。

slice(1,4) 提取原数组中的第二个元素开始直到第四个元素的所有元素 （索引为 1, 2, 3的元素）。

如果该参数为负数， 则它表示在原数组中的倒数第几个元素结束抽取。 slice(-2,-1)表示抽取了原数组中的倒数第二个元素到最后一个元素（不包含最后一个元素，也就是只有倒数第二个元素）。

如果 end 被省略，则slice 会一直提取到原数组末尾。

如果 end 大于数组长度，slice 也会一直提取到原数组末尾。

const aP = document.getElementById("#wrap");
let x = [].slice.call(ap);
//在ES6...扩展运算符之前常用以上办法将类数组变成数组
