# 算术运算符

# 1.定义变量的几个注意事项

同一个作用域下，let的变量名不能相同

对象的.操作添加属性等是不需要也不能使用let的

一个let可以用,隔开定义多个变量

# 2.算术运算符

+、-、*、/、%

小数运算，浏览器得到的值和实际运算所得的值有偏差

使用算数运算符时，会出现隐式类型转换

  1. # +运算符

    +两边任意一边有字符串，那么都会进行隐式字符串转换步骤，再进行拼接字符串操作

    let a = "10";
let b = 20;
console.log( a + b );
//最后控制台结果是1020；

    布尔值和数字运算时，会转换成 1 0

    let a = true;
let b = 20;
console.log( a + b );
//最后控制台结果是21；

  2. # -、*、/、%运算符

    -、*、/、%都会把其他数据类型变成数字再进行运算

    出现不合理的运算之后，会出现NaN(not a number)，注意NaN属于number类型

# 3.自增和自减

  1. 自增++

    ++在前，先进行自增运算，再进行其他运算

    ++在后，先进行自减运算，再进行其他运算

  2. 自减--

    同理

  3. 自增和自减隐式转换

    ++和--运算符，会强行将其他数据类型变成数字进行运算

# 4.模板字符串 ES6

单引号或者双引号描述的字符串，内部不能换行

`` 模板字符串可以，${}里面可以直接写字符串变量

let oWrap = document.getElementById("wrap");
let x = "goudan";
/*使用 字符串拼接模拟换行
oWrap.innerHTML = "<ul>"+
                    "<li>" +
                        "<p>" +
                            "<a href=\"\">" + x + "</a>"   +
                        "</p>" +
                    "</li>" +
                "</ul>";
*/
oWrap.innerHTML = `<div>
						<p>
							<i>${x}</i>
							<b>\$\{\}</b>
						</p>
				   </div>`;
