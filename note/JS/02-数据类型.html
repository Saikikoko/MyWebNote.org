# 数据类型

# 1. 数据类型初步了解

JavaScript里面数据可分为6大类：基础类（undefined、number、string、boolean、null），复杂类（object）。

  1. # undefined 未定义

    任何一个定义出来没有赋值的变量，默认数据都是undefined

  2. # number 数字

    10 10.1 0.3 .3

    NaN是一个不等于自身的数字

    Number.isNaN(..)是ES6新增的方法，用来检测一个值是否是NaN

    //检测数字是不是NaN
if(!Number.isNaN){
    Number.isNaN = function(n){
        return n !== n;
    };
}

  3. # string 字符串

    一对双引号 或者 一对单引号引起来的内容

    ES6中新的字符串定义方式 模板字符串 let x = `123`

    转义字符 \

    字符串的拼接

  4. # boolean 布尔值

    布尔值只包含两个 true false

    布尔值就是布尔值，不是0 和 1，0 和 1 是数字，和布尔值没有直接的联系

  5. # null 空

    空的对象指针，具体的应用倒是没什么，但是开发标准上来讲，在定义变量的时候，假设暂时不需要赋值，但是后面会表示一个对象，请给它赋初始值null。

  6. # symbol

    ES6新增的一种数据类型 通常用来给对象添加一个独一无二的属性属性

    let x = {
    name : "阿飞",
    age : 18
};
x["age"] = 80;//会覆盖18的值
x[ Symbol("age") ] = 88;//不会覆盖前面的18
console.log(x[ Symbol("age") ] );//取不到值，可以先定义变量y存取Symbol("age"),再用y取值
let y = Symbol("age");
x[ y ] = 99;

  7. # object 对象 引用数据类型

    对象包含很多很多种类，比如：数组，{}，内置对象，节点对象、函数 等等。

    不管什么对象都可以拥有很多属性（其他的基础数据类型除了JS规定的属性外，不能再添加独有的属性），这些属性就相当于一个隶属于该对象的变量，也可以用来存储任意数据类型的值。

    数组 [ ]一种特殊的对象，数据的组合，可以使用 [数字下标] 来取值，下标从0开始计数；

    1. JavaScript中，数组可以容纳任何类型的值

    2. 数组声明不需要预先设定大小

    3. 数组也是对象，因此也可以包含字符串键值和属性（但是并不计算在数组长度内）

    4. 如果字符串键值能够被强制类型转换为数字的话，就会被当作数字索引处理

    let x = ["dog","cat","pig"];
console.log(x[1]);//"cat"

var a = [];
a[0] = 1;
a["foo"]=2;
a.length;//1
a.foo = 2;
a.["foo"] = 2；

a["12"] = 12;
a.length//13

    {} 空对象，除了继承JS原本的对象描述外，什么都没有的对象；

我们可以使用.操作符或[]操作符去访问对象的属性

.a操作成为属性访问，["a"]称为键访问

这两种方式的区别：

.操作符要求属性名满足标识符的命名规范，而[..]语法可以接受任意UTF-8/Unicode字符串作为属性名

访问对象为定义的属性时，返回undefined

//json对象
let x = {
  name : "huadao",
  age : 20,
  height : 100
};
console.log(x.name);
//json属性和键值成对出现

/*两种对象取值的方式*/
console.log(x.name);//使用 .
console.log(x["name"]); 
/*当用 . 去取某一个对象的属性的时候，我们基本上都可以使用 [ ]的方式来代替,[]里面是字符串或者代表字符串的变量*/
let x = {
  name : "huadao",
  age : 20,
  height : 100
};
let a = "name";
console.log(x[a]);

/*特殊情况，对象中有数字属性*/
let x = {
    999 : 889,
    555 : 789,
    name : "王师傅",
    age : 18
};
 console.log( x[999] );//只能使用[]取值

可计算属性名

ES6中增加了可计算属性名，可以在声明形式中使用[]包裹一个表达式来当作属性名：

var a = "foo";

var obj = {
    [ a + "bar" ]: "hello",
    [ a + "baz" ]: "world"
}
obj["foobar"];//hello

# 3. typeof操作符

通过typeof可以用来快速的判断数据的类型，但是注意typeof得到的是一个字符串，也并不是准确的表示该数据的类型，只是方便我们去认识而已。

typeof undefined === "undefined";//true
typeof true === "boolean";//true
typeof 42 === "number";//true
typeof "42" === "string";//true
typeof {life:42} === "object";//true
typeof Symbol() === "symbol";//true

两种特殊情况:

typeof null === "object";//true

正确检测null的方式

var a = null;
(!a&&typeof a === "object");//true

还有一种情况

typeof function a(){} === "function";//true

除了function其他对象返回的结果都是object

typeof对已声明未赋值的变量和未声明的变量返回的都是undefined
