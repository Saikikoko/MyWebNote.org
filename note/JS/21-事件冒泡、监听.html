# 事件冒泡

子元素触发了事件，如果父元素也有同类型的事件，则父元素的事件也会触发

//box2是box1的子元素
box1.onclick = function(){
    console.log(1);
}
box2.onclick = function(){
    console.log(2);
}
//当点击box2时，会打印2 1

# 如何阻止事件冒泡

冒泡触发顺序

​ 从子元素开始到document

event.stopPropagation();//主流浏览器 ie8及以下不兼容

IE浏览器阻止冒泡的方法

event.cancelBubble = true

兼容性写法

event.stopPropagation?event.stopPropagation():(event.cancelBubble = true)

# 事件委托

利用事件冒泡的机制 ，对于重复性的事件添加，我们可以使用事件委托来处理

e.target得到当前触发事件的对象

oWrap.onclick = function(e){
    e = e || window.event;
    e.target = e.target || e. srcElement;
    if(e.target.tagName.toLowerCase() === "li"){
        console.log(e.target.innerHTML);
    }
}

IE8及以下不兼容e.target，它使用e.srcElement

# 事件的监听

# DOM 0级事件

box.onclick = function(){
   console.log(1); 
}
box.onclick = function(){
   console.log(2); 
}
//只打印2

DOM 0 级事件 后面的事件会把前面的事件覆盖

# 事件监听 DOM 2级事件

addEventListener

box.addEventListener("click",function(event){

});

  • 参数1 事件类型
  • 参数2 回调方法
  • 参数3 捕获|冒泡
    • true 捕获
    • false 冒泡

this指向触发事件的元素

兼容性

IE8及以下不支持addEventListener

IE10及以下可以使用attachEvent监听事件

  • 事件类型前必须加on，如onclick

  • 该方法的this指向window，不再指向触发事件的元素

  • 该方法没有事件捕获 只有冒泡

监听事件的触发机制默认是冒泡

捕获的事件触发顺序和冒泡相反

# 事件的取消

removeEventListener()

  • 参数1 与监听事件对应的事件类型
  • 参数2 与监听事件对应的回调函数名
  • 参数3 与监听事件对应的捕获|冒泡方式

IE8及以下

detachEvent()

  • 参数1 与监听事件对应的事件类型
  • 参数2 与监听事件对应的回调函数名

# 阻止事件的默认行为

DOM 0 级在事件里面直接return false

document.oncontentextmenu=function(event){
	return false;
});

DOM 2级

  • 主流浏览器 event.preventDefault()
document.addEventListener("contentextmenu",function(event){
	event.preventDefault();
});
  • IE event.returnValue = false
document.attachEvent("contentextmenu",function(event){
	event.preventDefault();
});

# 鼠标滚轮事件

  • IE和谷歌 mousewheel
    • event.wheelDelta
      • -120 下滚
      • 120 上滚
  • 火狐 DOMMouseScroll 只能通过二级事件监听
    • event.detail
      • 3 下滚
      • -3 上滚

onmousewheelIE和谷歌有，默认值为null，火狐没有，值为undefined

if(document.createElement("div").onmousewheel === null){
    console.log('谷歌')
}else{
    console.log('火狐')
}

利用该特性可写滚轮事件监听的兼容

# 兼容版的事件监听

function addEvent(el,type,cb,bool) {
    //判断火狐浏览器
    if( type ==='mousewheel' && el.onmousewheel === undefined ){
        type = 'DOMMouseScroll'
    }
    var callBack = function (e) {
        //兼容IE
        e = e || window.event;
        e.wheelDetail = e.wheelDelta / 120 || e.detail / -3;

        if(!!bool){
            e.preventDefault? e.preventDefault() : e.returnValue = false
        }
        //修复this指向问题
        cb.call(this,e)
    }
    //判断是否IE
    if( el.addEventListener ){
        el.addEventListener(type,callBack)
    }else{
        el.attachEvent("on"+type,callBack)
    }
    //返回回调函数 用于取消事件行为
    return callBack
}
//移出事件
function removeEvent(el,type,cb){
    if( type ==='mousewheel' && el.onmousewheel === undefined ){
        type = 'DOMMouseScroll'
    }
    el.removeEventListener?el.removeEventListener(type,cb):el.detachEvent("on"+type,cb)
}

//一次性事件
function onceEvent(el,type,cb){
    var fn = addEvent(el,type,function(e){
       cb.call(el,e);
       removeEvent(el,'click',fn);
    });
}
