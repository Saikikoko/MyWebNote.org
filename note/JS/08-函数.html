# 函数

# 定义一个函数

function a(){
    //code
}
a();//写在函数定义前或后都可以
//或者
let a = function(){
    //code
}
a();//只能写在函数定义后

区别在于，第一种定义方式，可以在后面定义前面调用，第二种定义方式，必须先定义后面再调用。

函数表达式

通常我们需要定义函数之后，才能用 函数名()的形式执行，

函数表达式可以直接在后面加小括号自执行执行。

如果function是声明中的第一个词，那么就是一个声明，否则就是一个函数表达式。

哪些写法能将函数变为函数表达式：

let a = function(){}//let a = function(){}();
(function(){})//(function(){})();
+function(){}//+function(){}();
-function(){}//-function(){}();
~function(){}//~function(){}();
!function(){}//!function(){}();

# 参数

实参、形参、不定参*

实参：函数自执行时，小括号里面传入的数据；

形参：函数定义时，小括号里面写的变量名；

不定参：函数内部的伪数组，用于存储着所有的实参集合；

# this

function a(){
    console.log(this);
}
console.log(a === window.a);//true 在全局作用域定义的具名函数就是window对象的一个同名方法
---------------------------------------------------------------------------------
var a = function (){
    console.log(this);
}

console.log(a === window.a);//true  var在全局作用域声明的变量就是window对象的一个同名属性
----------------------------------------------------------------------------
let a = function (){
    console.log(this);
}
console.log(a === window.a);//false let声明的全局变量不是window的属性
a(); //函数这样自执行，this指向window


let obj = {
    name : "阿飞",
    x : function(){
        console.log(this);
    }
};

obj.x(); //对象方法自执行，this指向对象

# return

函数的默认返回值是undefined

return立即结束函数，不再往下执行，（区别理解关键词break continue）

return可有可无，根据需求。

function(a,b){
    return a*b;
}
/*不写return默认undefined，也可以自己规定返回值*/

return可以返回任意数据类型的数据

# 作用域简单认识

if(false){
    var b = 10;
}
console.log(b);/*undefined*/
/*ES6*/
if(false){
    let b = 10;
}
console.log(b);/*报错*/

ES5中的var作用域只有函数的作用域、script作用域

ES6中let关键词定义的变量，起作用的范围是包含这个变量最近的{ }

for循环小括号中let关键词定义的变量的作用域包含{}大括号的作用域
