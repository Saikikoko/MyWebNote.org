# npm常用命令

  • npm init项目初始化

    初始化项目名称、版本号、项目描述等，不修改直接回车至，确认输入yes--npm init -y

  • npm i安装模块

    • -global简写-g 全局安装 在任意路径都可以运行模块
    • --save简写--S 生产环境所使用的模块 dependencies
    • --save-dev简写-D 开发环境所使用的模块 devDependencies
    • 模块@版本号 可以指定安装的版本号

  • npm uninstall删除模块 简写 npm un[包名]

    • -global简写-g 全局安装 在任意路径都可以运行模块
    • --save简写--S 生产环境所使用的模块 dependencies
    • --save-dev简写-D 开发环境所使用的模块 devDependencies

  • npm update [包名] 升级模块

    • -global简写-g 全局安装 在任意路径都可以运行模块
    • --save简写--S 生产环境所使用的模块 dependencies
    • --save-dev简写-D 开发环境所使用的模块 devDependencies
    • 模块@版本号

  • npm list列出安装的模块列表

  • npm publish上传编写的模块至npm官网
