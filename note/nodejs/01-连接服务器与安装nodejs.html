# 安装nodeJS

连接服务器与安装nodejs

mac使用自带终端

window使用Xshell第三方工具，下面步骤可以省略

mac：

#终端输入以下命令 回车（输入本机密码）
sudo su -

#然后输入 ssh -p 端口号（ssh端口号是22）服务器用户名（默认root）@服务器公网ip 例如：
ssh -p 22 root@47.92.153.88
#然后回车 输入yes确认 回车
#然后输入服务器密码
#然后看到类似：[root@iz8vb34affxrx2bxh3qwzlz -]# 表示登录成功

# 安装nodejs

# 源码安装

  1. 安装
#复制回车就行 v10.13.0代表版本号
wget https://nodejs.org/dist/v10.13.0/node-v10.13.0-linux-x64.tar.xz
#回车，等待安装成功
  1. 解压
#然后执行下面的命令进行解压
tar xvf node-v10.13.0-linux-x64.tar.xz
  1. 移动
#mv命令移动并改名
mv node-v10.13.0-linux-x64 /usr/local/node
  1. 配置命令
#先使用 cd命令切换到安装目录
vi ~/.bash_profile
#在-/.bash_profile 回车代开的文件里按 i 编辑，然后在文件的export PATH前一行添加
PATH=$PATH:/usr/local/node/bin
#然后退出保存文件 :wq
#运行
source ~/.bash_profile

#node -v npm -v 车看node和npm命令是否配置成功

# 使用包管理器安装

  1. 安装npm

    yum install npm

  2. 使用npm安装nodejs版本控制器 n

    npm i n -g

  3. 使用n命令安装nodeJS

    n stable#安装node稳定版
n latest#安装node最新版
n 10.0.1#安装node指定版本

  4. 使用n命令删除某个版本的node

    n rm 10.1.0

  5. 使用n命令选择node版本

    n #输入n回车，使用方向键选择

# 安装进程管理器

  1. 安装pm2

    npm i pm2 -g

  2. 通过pm2来启动进程或关闭进程

    pm2 start xxx.js[name] #启动
pm2 delete [id] #关闭进程
