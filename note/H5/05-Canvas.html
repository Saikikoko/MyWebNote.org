# Canvas

基本使用

<body>
	<canvas width=“” height=“”></canvas>
    <!--画布大小默认300*150-->
    <script>
        let oC = document.querySelector('canvas')

        //想要在canvas标签上绘制图形，必须要在2d环境下绘制
        //一般是绘制平面图形
        let cxt = oC.getContext('2d');
        cxt.rect(100,50,100,50);//矩形 x y 宽100 高50
        cxt.storke();//绘制矩形
    </script>
</body>

注意不能使用改变标签样式的方式（style）改变画布的大小width和height，否则画出的图形会被放大缩小，从而失真，要改变画布大小，必须直接修改标签属性

# 绘制方式

cxt.storke()绘制空心图形，默认边框黑色

cxt.fill()绘制实心图形，默认填充黑色

# 绘制样式属性

fillStyle()填充颜色

strokeStyle()触笔颜色

lineWidth触笔粗细 默认是2px

# 绘制矩形

cxt.rect(x,y,w,h) 不会直接显示，需要和绘制方式配合使用

cxt.storkeRect(x,y,width,height)绘制空心矩形

cxt.fillRect(x,y,width,height)绘制实心矩形

cxt.clearRect(x1,y1,x2,y2)清除范围内的矩形（x1,y1)~(x2,y2)

# 绘制线条

cxt.moveTo(x,y)起始点

cxt.lineTo(x,y)终止点

beginPath()使线条之间相互独立

closePath()闭合线条，回到起始点

cxt.translate()改变坐标原点
