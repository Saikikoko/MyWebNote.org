# H5文件下载和上传

# 文件下载

# a标签

a标签，压缩文件和程序默认点击可以下载，图片和文本需要添加download属性

<a href='./1.txt' download>下载文字</a>

# button按钮模拟a标签

btn.onclick = function(){
    const link = docuemnt.createElement('a')
    link.href = './1.txt'
    link.download = '文件.txt'
    link.click()
}

function download(filename,content){
    const blob = new Blob([content],{type:'application/msword'})
    const url = window.URL.createObjectUrl(blob)
    const link = document.createElement('a')
    link.href = url
    link.download = filename + '.doc'
    link.click()
}

# 文件上传

表单控件或H5拖拽+FormData对象实现文件上传功能

思路：

  1. 利用表单控件或H5拖拽获取文件信息
  2. 使用FormData携带数据

# 表单控件实现上传

php:

<?php
$myFile = $_FILES['file'];
$myName = $myFile['name'];
$myPath = 'upload/'.$myName;
move_uploaded_file($myFile['tmp_name'],$myPath);

?>

js:

const oInp = document.querySelector('input')
oInp.onchange = function () {
    if(!this.value)return;//判断是否选择了文件
    const oFile = this.files[0]
    uploadFile(oFile)
}

function uploadFile(file) {
    //图片预览
    const blob = new Blob([file])
    const url = window.URL.createObjectURL(blob)
    const image = new Image()
    image.src = url
    image.width = 200
    image.onload = function () {
        document.body.appendChild(image)
    }

    //上传
    const xhr = new XMLHttpRequest()
    xhr.open('post','./upload.php',true)
    const fd = new FormData()
    fd.append('file',file)
    xhr.send(fd)
}

# H5拖拽实现上传

js:

const oWrap = document.getElementById("wrap")
oWrap.ondragover = function (ev) {
    ev.stopPropagation()
    ev.preventDefault()
}

oWrap.ondrop = function (ev) {
    ev.stopPropagation()
    ev.preventDefault()
    const oFile = ev.dataTransfer.files[0]
    uploadFile(oFile)
}

# 监听上传进度

通过onprogress事件监听事件对象的loaded和total属性

仿github访问进度条

xhr.upload.onprogress = function(e){
    const scale = e.loaded / e.total
    p.style.width = scale * window.innerWidth + 'px'
}

# 同源同域名下跨文档操作

同源指同一个服务器，同域指同一个端口

有两种方式：

  1. iframe内页:

  • 父页面操作子页面：contentWindow

  • 子页面操作父页面：window.top(找到最顶级的父页面)/parent(第一父页面)

父页面：

<body>
<h1>这是父页面</h1>
<input type="button" id="btn" value="点击子页面的背景颜色"><br>
<iframe src="2iframe.html" frameborder="0"></iframe>

<script>
    btn.onclick = function () {
        const parent = document.querySelector('iframe')
        parent.contentWindow.document.body.style.background = 'hotpink'
    }
</script>
</body>

子页面：

<body>
<h2>这是子页面</h2>

<input type="button" id="btn" value="点击改变父页面（1iframe.html）的背景">
<script>
    btn.onclick = function () {
        console.log(parent)
        parent.document.body.style.background = 'yellowgreen';
    }
    // parent -> 父页面
    // window -> 当前网页
    // top -> 顶层页面
</script>
</body>
  1. 新窗口页：
  • 父页面操作子页面：window.open
  • 子页面操作父页面：window.opener

父页面：

<body>
<input type="button" id="btn1" value="点击此按钮打开一个新窗口">
<input type="button" id="btn2" value="点击去改变打开的窗口的背景颜色">
<script>
    let opener;
    btn1.onclick = function () {
        opener = window.open('4iframe.html');
    }
    btn2.onclick = function () {
        opener.document.body.style.background = 'skyblue';
    }
</script>
</body>

子页面：

<body>
    <h4>这是子窗口</h4>
    <input type="button" id="btn" value="点击此按钮去改变父页面3iframe.html">
    <script>
        btn.onclick = function () {
            window.opener.document.body.style.background = 'red';
        }
    </script>
</body>
