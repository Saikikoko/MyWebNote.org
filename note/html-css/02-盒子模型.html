# 盒子模型

content：内容

padding：内边距

border： 边框

margin：外边距

# 盒子命名：

  • id：一个标签只能有一个ID，其他盒子的ID不能和这个一样
  • class：一个标签可以有很多的class，其他人的class可以和这个class重复

命名规则：

​ 字母 数字 -

​ 不能以数字 或-开头

​ 严格注意大小写，没有特殊需要请使用小写

​ 见名知义

盒子模型：

因为很多元素有默认的边距，然后我们不需要这个边距，所以我们要干掉这些默认的边距 */
/* *{ margin:0; padding:0;}  终极大招，全部元素margin归0，padding归0 */
 body{ margin:0; }/*哪个标签需要初始化就写哪个，通常推荐这种写法，不会浪费性能*/

# 内边距padding

填充物，会有背景，但是内容不会显示在padding区域，padding不可以是负值

/*padding:20px; */               /*一个值： 上下左右 */
/*padding:10px 20px; */          /*两个值： 上下  左右 */
/*padding:10px 20px 30px;*/      /*三个值： 上   左右   下  */
/*padding:10px 20px 30px 40px;*/ /*四个值： 上  右  下  左*/
/* 支持分解方位的属性  -top -bottom -left -right */
padding-right:20px;
padding-left:5px;

# 边框border

border:5px solid #000;  /* 四条边框边框的粗细  线的类型  颜色 */
/*修改四条边框的粗细或类型或颜色*/
border-width:5px
border-width:5px 10px;
border-width:5px 10px 20px;
border-width:5px 10px 20px 30px;
/*像padding一样有一个值、两个值、三个值、四个值的写法*/
border-style:solid dotted dashed double;/*颜色和样式也可以写四个值*/
border-color:red green yellow black;
/*修改某一边框的粗细、类型、颜色*/
border-right:5px solid pink;
/* 要改某一边边框的粗细或类型或颜色，只能单独写分解属性去改 */
border-right-width: 15px;   /* -右-粗细 */
border-bottom-style: dotted;  /* -下-线类型 */
border-left-color: #f00;  /* -左-颜色 */

margin支持负值

/*margin: 20px;*/
margin: auto;/*在设置了盒子width之后，可以实现盒子居中（块级盒子且在文档流中）*/
# 外边距合并

外边距合并准则

  • 正正取最大
  • 正负值相加
  • 负负取绝对值最大

父子级关系

<head>
    <style>            
        body,p,ul{margin:0; padding:0;}
        /*
        第一个子元素的margin会传递给父级
        需要通过触发BFC来解决，后面再讲
        */
        div{
            width:200px;
            height:200px;
            background:red;
        }
        p{
            width:50px;
            height:50px;
            background:pink;
            margin-top:20px;
        }
        ul{
            margin-top:20px;
        }
    </style>
</head>
<body>       
    <div>
        <p>123</p>
        <ul>aaa</ul>
    </div>
</body>

兄弟级

<head>
    <style>            
        body,p,ul{margin:0; padding:0;}
        /*
        上下相同层级的元素，上下外边距相遇时，只会取最大的
        触发BFC解决
        */

        div{
            width:100px;
            height:100px;
            background:red;
            margin-bottom:50px;
        }

        p{
            width:50px;
            height:100px;
            background:pink;
            margin-top:20px;
        }
    </style>
</head>
<body>       
    <div></div>
    <p></p>
</body>
