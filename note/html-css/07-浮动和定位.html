# 浮动和定位

# 浮动

定义：使元素脱离文档流，按照指定方向（左右）移动

浮动元素不再占据原来的位置，贴左贴右排布（贴父级的左边/右边）

不会超过前面没有浮动的块级元素

浮动元素，会挤开其他元素的文字布局

浮动元素 向左/向右 走遇到父级边框会停，遇到浮动元素的边缘也会停

当多个浮动元素排布的时候，父级宽度不够，横排显示不了的元素会往下掉

# 浮动元素的特性

  1. 块级（横排显示）
  2. 内联支持宽高
  3. 默认内容撑开宽度
  4. 支持margin 不支持margin:auto
  5. 脱离文档流
  6. 提升层级半层

# 清除浮动（常用）

  1. 隔板法 (兄弟元素被浮动元素覆盖+父级元素高度坍塌)

想要避免浮动后面的元素往上跑，可以在浮动元素和往上移动的元素之间插入一个块级元素，并且给该元素写上样式clear:both

  1. 给height（父级元素高度坍塌）

父级元素给高度height

  1. overflow:hidden（父级元素高度坍塌）

父级元素css样式添加overflow:hidden，原理是生成BFC环境（一旦生成BFC环境，父级元素会包裹所有的自己元素）

  1. 伪元素::after原理和隔板法一样（父级元素高度坍塌）

将浮动元素用一个父级标签包裹起来，给上类名clearfix

.clearfix::after{
    content:'';
    display:block;
    clear:both;
}

# 定位

脱离文档流

# position:relative相对定位

保留原来文档流的位置，可以使用方位词移动元素的位置，参照原文档流的位置

  • top
  • bottom
  • left
  • right

# position:absolute绝对定位

脱离文档流，参照物：最近的有定位父级或者HTML

# position:fixed固定定位

参照物：当前显示的窗口

# 定位的层级关系

两个定位元素发生重叠的时候，布局在后的显示在上面，可以通过修改z-index来改变显示顺序
