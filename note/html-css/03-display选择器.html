# display选择器

# 盒子模型补充

内容可以撑开盒子

有可能需要内容超出盒子范围，

但是一般正常布局情况下，需要控制好盒子大小，使他包围内容

当内容超出盒子的时候，可以设置overflow属性

overflow: hidden;//超出隐藏
overflow: visible;//默认显示
overflow: auto;//超出显示滚动条
overflow: scroll;//不管超不超出都有滚动条
/*拆分写法*/
overflow-x: auto;
overflow-y: hidden;

# 简单的样式初始化

body,dl,dd,p,h1,h2,h3,h4,h5,h6{margin:0;}
ol,ul,li{margin:0;padding:0;list-style:none;}
img{border:none;}

# 元素类型

# 块级元素 div h1~h6 p ol ul li dl dd dt

独占一行

默认支持宽高

支持上下外边距

# 行内元素 span a b i strong em img

不会独占一行

默认不支持宽高

不支持上下外边距

# 行内元素和块级元素之间的相互转换

display: block拥有块级元素特性

display: inline拥有行内元素特性

# 行内块元素

支持宽高

不独占一行

支持上下margin

# display:none

隐藏，不占据空间
