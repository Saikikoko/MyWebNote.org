# 选择器-伪类-伪元素

# 选择器

# 子代选择器

> 特殊的后代选择器，但是只能选择到子代

# 相邻兄弟选择器

**+**只能选择到相邻的兄弟元素

# 普通兄弟选择器

**~**可以向下选择到所有的兄弟元素

# 属性选择器

[attr] 选到有attr属性的标签

[attr=val] 选到标签属性attr=val的标签

[attr*=val] 选到attr属性有val字样的标签

[attr^=val] 选到attr属性由val开头的标签

[attr$=val] 选到attr属性由val结尾的标签

# css3新增选择器

:nth-child(值) 向下选取第几个子元素 值的取值如下

  • 任意整数
  • odd/even odd奇数 even偶数
  • 选到符合表达式的元素 a,b取任意实数 n的取值[0,无穷大)

:nth-last-child(值) 向上选取第几个子元素 取值同上

:nth-of-type(值) 向下匹配同类型中的第n个同级兄弟子元素 取值同上

:nth-last-of-type(值) 向上匹配同类型中的第n个同级兄弟子元素 取值同上

:empty 选到没有(子元素/文本)的元素

:not(选择器) 取反选择器

:first-child 选择第一个子元素

:flast-child 选择最后一个子元素

# 伪类选择器

:link用于选择未被访问的a标签 默认

visited选择被访问过的a标签

:hover 用于选择鼠标指针浮动在上面的元素，hover和link一起使用，hover需要写在link前面

：active 选择鼠标点击的元素

# 伪元素

::before 在被选择元素的内容前面插入内容

  • 默认为行内元素
  • content是必须的属性，用来激活伪元素

::after 在被选择元素的内容后面插入内容

  • 默认为行内元素
  • content是必须的属性，用来激活伪元素
