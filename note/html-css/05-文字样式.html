# 文字样式

# 字体

**color **字体颜色

font-size 字体大小 浏览器默认字体大小16px 默认最小字体大小12px（12px以下值无效） 可以为0

font-weight 粗细 （100~900 整百数，默认400/normal lighter bold bolder）

font-family 字体种类

font-style 风格斜体

  • italic
  • oblique
  • normal

line-height 行高

  • px
  • % 相对于自身字体大小font-size的百分比
  • 数字 相对于自身字体大小的倍数

font-variant 小型大写英文字母

  • small-caps
/*
    font复合写法
    斜体 小型大写 粗细 字体/行高 种类
    * 可以省略其中某几个，但是不能改变顺序

    letter-spacing 字间距
*/
p{
    /* font: italic normal 400 14px / 30px "微软雅黑","楷体"; */
    font: 700 12px / 30px "微软雅黑";
}

# 文本

text-transform

  • uppercase 文本中的小写英文字母变成大写
  • lowercase 文本中大写字母变成小写
  • capitalize 文本中的英文单词首字母大写

text-align

  • left左对齐
  • right右对齐
  • center 居中对齐
  • justify 两端对齐 单行文本不生效，要生效可以用伪元素::after在文本后产生一行空的文本

**text-indent**文本缩进

  • px
  • em 等于自身字体大小
  • % 相对于父级元素的宽度

text-decoration

  • underline 下划线
  • line-through 删除线、中划线
  • overline 上划线
  • none

word-break：break-all；强制打断单词 超出容器强制换行

white-space

  • normal换行
  • nowrap 强制段落不换行

letter-spacing 设置字母间隙

  • px

word-spacing设置单词间的间隙

  • px

text-overflow文本超出处理方式

  • ellipsis超出的文本处理成省略号

单行文本省略

/*单行文本超出容器，用···替代文本*/
p{
    overflow：hidden；
    white-space：nowrap；
    text-overflow：ellipsis；
}

# 继承属性

当标签没有设置 某些样式 的时候，他会继承自他父级的样式

以上文字和文本属性除了white-space外都可以被继承

还有list-style和cursor

有些元素本身具有独特的样式，他会使用自己的样式，而不是继承下来的样式

如a标签不会继承父级的color和text-decoration，标题标签不会继承字体大小和粗细

默认不继承的属性需要继承的话 可以使用inherit值来继承，但是ie浏览器不支持这个值
