# 变量

使用@声明变量

  1. 属性值变量
@color = '#ccc';
.box{
    background: @color;
}
  1. 选择器变量
@Wrap:wrap;
@Selector: #wrap;
.@{Wrap}{//变量名必须使用{}包裹
	color:pink    
}
@{Selector}{
    
}
#@{Wrap}{
    
}
  1. 属性变量
@borderStyle: border-style;
@Soild:solid;
#wrap{
  @{borderStyle}: @Soild;//变量名 必须使用大括号包裹
}

/* 生成的 CSS */
#wrap{
  border-style:solid;
}
  1. URL变量
@images: "../img";//需要加引号
body {
  background: url("@{images}/dog.png");//变量名 必须使用大括号包裹
}

/* 生成的 CSS */
body {
  background: url("../img/dog.png");
}

# 混合mixin

//混合
.border(){ //等价于.border
    border:1px solid pink;
}
.box{
    width:100px;
    height:100px;
    .border();//等价于.border
}
//混合带参数
.border-01(@border-width:10px){
    border:@border-width solid pink;
}
.box2{
    width:100px;
    height:100px;
    .border-01(100px);
    //.border-01();不传值默认10px
}

//匹配模式
.pos(t){
	top: 0;
    left:0;
}

.pos(b){
    bottom:0;
    left:0;
}

.box3{
    position:absolute;
    .pos(b);
}

要点：

  • . 与 # 皆可作为 方法前缀。
  • 方法后写不写 () 看个人习惯。

默认参数方法

  • Less 可以使用默认参数，如果 没有传参数，那么将使用默认参数。
  • @arguments 犹如 JS 中的 arguments 指代的是 全部参数。
  • 传的参数中 必须带着单位。

# 运算

@wd:100px;

.box{
    width:@wd - 20;
}
  • 加减操作以第一个运算单位为准
  • 乘除操作单位要一致
